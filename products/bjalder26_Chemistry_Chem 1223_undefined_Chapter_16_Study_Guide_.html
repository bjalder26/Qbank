1) Which of the following Ka values would correspond to the strongest acid?
A) Ka = 1.25 x 10-4
B) Ka = 6.34 x 10-8
C) Ka = 8.09 x 10-5
D) Ka = 1.38 x 10-2*
Since Ka is essentially products over reactants, the highest Ka corresponds with the strongest acid.

2) Which of the following is the strongest Lewis acid (best at giving up H+ or binding OH-)?
A) Li+
B) K+
C) Na+
D) HF*
The strongest acid, will have the weakest conjugate base.  The conjugate bases of Na+, Li+, and K+ are all strong bases (NaOH, LiOH, and KOH).  Thus the answer is HF.

3) Which of the following has the strongest conjugate base?
A) HCN*
B) HI
C) HCl
D) HBr
The weakest acid will have the strongest conjugate base.  Out of all the choices, only HCN is a weak acid.

4) If a solution has a hydroxide ion concentration of 4.10x10-4 M, then what is the pH of the solution at 25°C?
A) -3.45
B) 2.44x10-11
C) 3.39
D) 10.6*
pOH = -log[OH-]
pOH = -log(4.10x10-4)
pOH = 3.39
pH = 14 - pOH
pH = 14 - 3.39
pH = 10.6

6) Which of the following pKa values would correspond to the strongest acid?
A) pKa = 9.64
B) pKa = 5.79
C) pKa = 3.79
D) pKa = 1.51*
E) None of the above.
The lowest pKa corresponds to the strongest acid.  Possibly the easiest way to remember this is that at the half-equivalence point pH = pKa.  The strongest acid would have the lowest pH, thus the lowest pKa.

A) 4.50
B) 9.91
C) 1.23x10-10
D) 4.09*

I 0.541   0 0
C - x   + x + x
E 0.541 - x   x x

0.541 x 4.38x10-4= X2

pH = 14 - pH = 14 - 1.81 = 12.2

9) What is the conjugate acid of HSO4-?
A) SO4OH
B) SO42-
C) H2SO4*
D) None of the above.
To find the conjugate acid of HSO4-, add a H+ ion (or remove OH-).  This will give you H2SO4, which has 1 more hydrogen and a charge 1 higher.

10) Determine the [H3O+] in a 0.270 M HClO solution. The Ka of HClO is 2.9 × 10-8.
A) 0.000106 M
B) 7.83x10-9 M
C) 8670 M
D) 0.0000885 M*
Since HClO is a weak acid, we need a RICE table, and acid dissociation constant equation.
R HClO ⇄ H+ +  ClO-
I 0.270 0 0
C - x + x + x
E 0.270 - x x x

Ka = 2.9 × 10-8 = [H+][ClO-]/[HClO]
2.9 × 10-8 = x2/(0.270 - x), small x assumption is valid because 0.270/(2.9 × 10-8) = 9.31x106
2.9 × 10-8 = x2/0.270
0.270 x 2.9 × 10-8 = x2
x = √(0.270 x 2.9 × 10-8)
[H3O+] = x = 0.0000885 M

13) Which of the following is the strongest acid?
A) H2PO4-
B) HPO42-
C) H3PO4*
D) PO43-
Oxyacids become stronger acids the more oxygens they have.  Also, the more positively charged an acid is (least negatively charged), the stronger the acid will be.

15) Determine the pH in a 0.906 M HSO4- solution. The Kb of SO42- is 8.33 × 10-13.
A) 1.01*
B) 2.04
C) 13.0
D) None of the above.
Since HSO4- is a weak acid, we need a RICE table, and acid dissociation constant equation.
R HSO4- ⇄ H+ +  SO42-
I 0.906 0 0
C - x + x + x
E 0.906 - x x x


Ka = 10-14/8.33 × 10-13 = 0.0120
0.0120 = [H+][SO42-]/[HSO4-]
0.0120 = x2/(0.906 - x), small x assumption is not valid because 0.906/(1.2 × 10-2) = 75.5
0.0120 = x2/0.906
(0.906 - x) x (0.0120) = x2
0.0109 - 0.0120x = x2
0 = x2 + 0.0120x - 0.0109
x = 0.0985
pH = -log[H3O+]
pH = -log(0.0985)
pH = 1.01

16) Which of the following chemicals are amphoteric?
A) NH4+
B) Ca(OH)2
C) HPO42-*
D) NO3-
Amphoteric (amph- like amphibian, means both) means that it can act as either an acid or a base.  What you should be looking for is a chemical with an acidic hydrogen (typically it will have the hydrogen listed first or a carboxylic acid group -COOH), that can also accept an additional hydrogen (e.g. water or negatively charged).  Thus HPO42- can be classified as amphoteric.

18) What is the conjugate base of  HSO4-?
A) SO42-*
B) H2SO4
C) SO4OH
D) None of the above.
To get the conjugate base, you'd need to remove a H+ or add an OH-.  In this case, when we remove a H+ from HSO4- it gives us SO42-.

24) (SLO 4.2) Which of the following is a Brønsted–Lowry base?
A) H3PO4
B) Mg(OH)2*
C) CH4
D) Br2
E) None of the above.
H3PO4 is an acid.
Br2 will not dissociate in water, and is not an acid or a base.
CH4 does not have an acidic hydrogen (the C-H bonds won't break), and is not an acid or a base.
Mg(OH)2 is the only base.

25) (SLO 4.7) How many milliliters of 0.0852 M NaOH are required to titrate 30 mL of 0.145 M HBr to the equivalence point.
A) 17.6 mL
B) 5.11 mL
C) 51.1 mL*
D) 43.4 mL

26) (SLO 4.6) What is the [CH3CO2-]/[CH3CO2H] ratio necessary to make a buffer solution with a pH of 5.97? 
Ka = 1.8x10-5 for CH3CO2H.
A) 17 : 1*
B) 0.060 : 1
C) 9.3 : 1
D) 20. : 1
pH = -log(Ka) + log(base/acid)
5.97 = -log(1.8x10-5) + log([CH3CO2-]/[CH3CO2H])
5.97 = -5.26 + log([CH3CO2-]/[CH3CO2H])
5.97 - -5.26 = log([CH3CO2-]/[CH3CO2H])
11.2 = log([CH3CO2-]/[CH3CO2H])
1011.2  = [CH3CO2-]/[CH3CO2H]
[CH3CO2-]/[CH3CO2H] = 17 : 1

28) (SLO 4.3) Identify the weakest acid.
A) H2SO3
B) They are all equal strength
C) H2SO4
D) H2SO2*
The oxyacid with the fewest oxygens will be the weakest acid, because it's conjugate base will be the least stable due to resonance.
