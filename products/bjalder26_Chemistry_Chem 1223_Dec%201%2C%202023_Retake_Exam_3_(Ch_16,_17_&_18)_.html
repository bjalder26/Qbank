1) What is the conjugate acid of OH-?
A) O2-
B) H3O+
C) H2O*
D) None of the above.
To find the conjugate acid of OH-, add a H+ ion (or remove OH-).  This will give you H2O, which has 1 more hydrogen and a charge 1 higher.

2) Which of the following is the strongest acid?
A) HBrO2
B) HBrO
C) HBrO4*
D) HBrO3
Oxyacids become stronger acids the more oxygens they have.  Also, the more positively charged an acid is (least negatively charged), the stronger the acid will be.

3) Which of the following pKa values would correspond to the strongest acid?
A) pKa = 7.32
B) pKa = 1.47*
C) pKa = 3.75
D) pKa = 6.60
E) None of the above.
The lowest pKa corresponds to the strongest acid.  Possibly the easiest way to remember this is that at the half-equivalence point pH = pKa.  The strongest acid would have the lowest pH, thus the lowest pKa.

4) Which of the following chemicals are amphoteric?
A) SO2-
B) H2O*
C) H3O+
D) NaOH
Amphoteric (amph- like amphibian, means both) means that it can act as either an acid or a base.  What you should be looking for is a chemical with an acidic hydrogen (typically it will have the hydrogen listed first or a carboxylic acid group -COOH), that can also accept an additional hydrogen (e.g. water or negatively charged).  Thus H2O can be classified as amphoteric.

5) Determine the pH of a 0.536 M HI  solution.
A) 13.7
B) 0.271*
C) 0.352
D) 1.87x10-14
Since HI is a strong acid, assume the concentration of HI is the hydrogen ion concentration.
pH = -log([H+])
pH = -log(0.536 M)
pH = 0.271

6) Determine the [H3O+] in a 0.866 M HC2H3O2 solution. The Ka of HC2H3O2 is 1.8 × 10-5.
A) 3.95x10-8 M*
B) 1.56x10-10 M
C) 1.22x105 M
D) 4.74x10-8 M
Since HC2H3O2 is a weak acid, we need a RICE table, and acid dissociation constant equation.
R HC2H3O2 ⇄ H+ +  C2H3O2-
I 0.866 0 0
C - x + x + x
E 0.866 - x x x

Ka = 1.8 × 10-5 = [H+][C2H3O2-]/[HC2H3O2]
1.8 × 10-5 = x2/(0.866 - x), small x assumption is valid because 0.866/(1.8 × 10-5) = 4.81x104
1.8 × 10-5 = x2/0.866
0.866 x 1.8 × 10-5 = x2
x = √(0.866 x 1.8 × 10-5)
[H3O+] = x = 3.95x10-8 M

7) Determine the pH in a 0.640 M H2SO3 solution. The Kb of HSO3- is 8.33 × 10-13.
A) 12.9
B) 1.09*
C) 1.81
D) None of the above.
Since H2SO3 is a weak acid, we need a RICE table, and acid dissociation constant equation.
R H2SO3 ⇄ H+ +  HSO3-
I 0.640 0 0
C - x + x + x
E 0.640 - x x x


Ka = 10-14/8.33 × 10-13 = 0.0120
0.0120 = [H+][HSO3-]/[H2SO3]
0.0120 = x2/(0.640 - x), small x assumption is not valid because 0.640/(1.2 × 10-2) = 53.3
0.0120 = x2/0.640
(0.640 - x) x (0.0120) = x2
0.00768 - 0.0120x = x2
0 = x2 + 0.0120x - 0.00768
x = 0.0819
pH = -log[H3O+]
pH = -log(0.0819)
pH = 1.09

12) You add 500 mL of a 0.110 M H2SO4 solution to 800 mL of a 0.180 M KOH solution.  What will be the pH of that solution?
A) 12.8*
B) 12.6
C) 1.16
D) 13.2
Molarity of strong acid:
500 mL x (1L/1000mL) x (0.11 mol/L) = 0.0550 mol
0.0550 mol / 1.30 L = 0.0423 M
Molarity of strong base:
800 mL x (1L/1000mL) x (0.18 mol/L) = 0.144 mol
0.144 mol / 1.30 L = 0.111 M

Since there is more strong base than strong acid, subtracting the molarity of the acid from the molarity of the base will tell us what the remaining molarity of strong base will be.
0.111 M  KOH - 0.0423 M H2SO4 = 0.0685 M KOH remaining
pOH = 1.16
pH = 14 - 1.16 = 12.8

13) You add 700 mL of a 0.160 M NaOH solution to 500 mL of a 0.110 M HNO2 solution (Ka for HNO2 is 4.5x10-4).  What will be the pH of that solution?
A) 3.05
B) 12.9
C) 12.7*
D) 1.24
Molarity strong base
700 mL x (1L/1000mL) x (0.16 mol/L) = 0.112 mol NaOH
0.112 mol NaOH / 1.20 L = 0.0933 M

Molarity weak acid
500 mL x (1L/1000mL) x (0.11 mol/L) = 0.0550 mol HNO2
0.0550 mol HNO2 / 1.20 L = 0.0458 M

Since there is more strong base than weak acid, the pH will be based on the remaining strong base.
Subtract the molarity of weak acid from the molarity of strong base to get the remaining strong base.
0.0933 M HBr - 0.0458 M CH3NH2 = 0.0475 M NaOH remaining
pOH = -log(0.0475)
pOH 1.32
pH = 14 - pOH = 14 - 1.32 = 12.7

14) What would be the pH of a solution formed by mixing 300 mL of 1.52 M H2CO3 with 300 mL of 0.318 M NaHCO3?  The Ka for H2CO3 is 4.5x10-7.
A) 5.67*
B) 6.35
C) 6.72
D) 7.03
In this case, you're mixing a weak acid with it's weak conjugate base.
This will create a buffer solution, for which we will use the Henderson–Hasselbalch equation to solve for pH.
First we need the moles of the acid and the base.
[H2CO3] = 300 mL x (1 L/1000 mL) x (1.52 mol/1L) = 0.456 mol
[NaHCO3] = 300 mL x (1 L/1000 mL) x (0.318 mol/1L) = 0.0954 mol*
pH = -log(Ka) + log(base/acid) = -log(4.5x10-7) + log(0.0954/0.456)
pH = 5.67
*since we were using the same volume, we could have just plugged in molarity (before mixing) as a short-cut, or we could have calculated millimoles by not dividing by 1000, as a different shortcut.

15) Determine the molar solubility of AgBr in a solution containing 0.120 M NaBr. Ksp (AgBr) = 7.7 × 10-13.
A) -0.0528 M
B) 6.42x10-12 M*
C) 1.28x10-11 M
D) None of the above.
R AgBr (s) ⇄ Ag+ + Br-
I   0 0.12
C   + x + x
E   x 0.12 + x 

 Ksp (AgBr ) = 7.7 × 10-13 = (x)(0.12+ x)
Since 0.12/(7.7 × 10-13) = 1.56x1011, the small x assumption is valid
7.7 × 10-13 = 0.12x
x = (7.7 × 10-13)/0.12 = 6.42x10-12 M

16) You add 400 mL of a 0.120 M HCl solution to 400 mL of a 0.120 M NH3 solution (Kb for NH3 is 1.8x10-5).  What will be the pH of that solution?
A) 2.98
B) 4.74
C) 9.26
D) 5.24*
In this problem, there is an equal amount of strong acid, and weak base.  This means the strong acid and strong base will both be completely consumed.  That will leave the conjugate weak acid.
HCl + NH3 ⇄ NH4+ + Cl- 
moles of NH4+ = 400 mL x (1L/1000mL) x (0.12 mol/L) = 0.0480
[NH4+] = 0.0480 / 0.800 L = 0.0600 M
  NH4+ + H2O ⇄ NH3 H3O+
beg 0.0600       
change - x   + x + x
end 0.0600 -x   x x

Ka = 10-14/ 1.8x10-5 = 5.56x10-10
Ka = 5.56x10-10 = x2/(0.0600 -x)
Since 0.0600 / 5.56x10-10 = 1.08x108, the small x assumption is valid
5.56x10-10 = x2/(0.0600)
(0.0600) x 5.56x10-10 = x2
√(3.33x10-11) = x  
x = 5.77x10-6
pH = 5.24

19) If you have a 1.0 L buffer solution with 0.10 M acetic acid (Ka 1.75x10-5) and 0.10 M sodium acetate, then what will be the pH after adding 0.0200 moles of solid NaOH (assume no change in volume)?
A) 4.76
B) 4.93*
C) 4.58
D) 5.07
Beginning moles:
1L x (0.10 mol acetic acid/L) = 0.10 mol acetic acid
1L x (0.10 mol sodium acetate/L) = 0.10 mol sodium acetate

Moles after adding 0.02 moles of NaOH:
0.02 moles of NaOH will react with 0.02 moles of acetic acid to produce 0.02 moles of acetate
0.10 mol acetic acid -0.02 mol = 0.0800 mol acetic acid
0.10 mol sodium acetate +0.02 mol = 0.120 mol acetic acid
pH = -log(1.75*10-5) + log(0.120)/(0.0800)
pH = 4.93

20) What would be the pH of a solution formed by mixing 400 mL of 1.41 M NH4Cl with 400 mL of 0.132 M NH3?  The Kb for NH3 is 1.59x10-5.
A) 10.2
B) 7.15
C) 9.20
D) 8.17*
In this case, you're mixing a weak acid with it's weak conjugate base.
This will create a buffer solution, for which we will use the Henderson–Hasselbalch equation to solve for pH.
First we need the moles of the acid and the base.
[NH4Cl] = 400 mL x (1 L/1000 mL) x (1.41 mol/1L) = 0.564 mol
[NH3] = 400 mL x (1 L/1000 mL) x (0.132 mol/1L) = 0.0528 mol*
Ka = 10-14/Kb = 10^-14/(1.59x10-5)
pH = -log(Ka) + log(base/acid) = -log(6.29x10-10) + log(0.0528/0.564)
pH = 8.17
*since we were using the same volume, we could have just plugged in molarity (before mixing) as a short-cut, or we could have calculated millimoles by not dividing by 1000, as a different shortcut.
